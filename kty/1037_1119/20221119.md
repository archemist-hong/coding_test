## 해설
- 약수 성질 활용
1. 약수들을 차례대로 나열
2. 양끝값을 곱해주면 실제값 도출

## 중요 포인트
- 약수 정렬 -> 정렬 수 양끝값씩 곱해주면 실제값이 나온다

## 정답코드
Success : 
``` python
a_number = int(input()) # 양수개수 받기 (필요없을듯)
a = list(map(int,input().split(" "))) # 1과 실제값을 제외한 약수 받기
a.sort() # 약수들 정렬
print(a[0]*a[-1]) # 약수 양끝값을 곱해주면 실제값 나옴

```
